Elementos basicos

  1. La etiqeuta "DOCTYPE" define la declaracion del documento como HTML
  2. La etiqueta "HEAD" contiene informacion adicional acerca de la pagina HTML
  3. La etiqueta "HTML" es el elemento principal de una pagina HTML
  4. La etiqueta "TITLE" es el nombre del titulo de la pagina
  5. La etiqueta "BODY" es el cuerpo de la pagina
  6. La etiqueta "h1" define un texto de tamaño titular
  7. La etiqueta "h2" define un texto de tamaño sub titular
  8. La etiqueta "p" define un texto de tamaño parrafo
